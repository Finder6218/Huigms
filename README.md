# 家居厨具进销存管理系统

  《家居厨具进销存管理系统》是一款集家居厨具进、销、存、订单等管理的软件，适用于各大中小型专卖店。   
  
软件在功能方面主要包括：   
基本信息：商品类别、商品信息、供商信息；    
订单管理：订单登记、已签收订单查看、付订单欠款、客户欠款查看；     
进销存管理：采购登记、采购退货、期间采购查询汇总、期间退货查询汇总、销售登记、期间销售查询汇总、库存明细；   
员工管理：员工信息、提成汇总。    
  
# 如何运行本系统?

本系统基于宏达数据库信息管理开发平台开发，下载后，双击文件夹中"dbimp.exe"文件即可完整运行对应的信息管理系统。
